# Prompts utilizados

## Chatbot: ChatGPT (GPT-4)

## Prompt 1 (único prompt utilizado):

"Tengo una página web con dos archivos: index.html y script.js. 
Necesito implementar la lógica para invertir el orden de una cadena de texto.
Por ejemplo: si introduzco 'AI4Devs' debe devolver 'sveD4IA'.

El archivo index.html debe tener:
- Un título principal que diga 'Reverse String'
- Un campo de texto (input) donde el usuario pueda escribir
- Un botón para ejecutar la inversión
- Un área donde se muestre el resultado

El archivo script.js debe contener:
- Una función que invierta una cadena de texto
- La lógica para obtener el valor del input, procesarlo y mostrar el resultado
- Manejo del evento del botón

Usa HTML5, CSS moderno con un diseño limpio y atractivo, y JavaScript vanilla (sin librerías externas).
El diseño debe ser responsive y usar una paleta de colores profesional."
